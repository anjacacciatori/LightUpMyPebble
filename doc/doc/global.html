ajax()

Does PUT and GET queries in order to obtain the state of the light or of a group or to set some values on them.

This function is basically composed by

    • A first part with a field
      • "url" : it indicates the url to which the request has to be done
      • "method" : it indicates if the query is a GET, PUT, POST or DELATE
      • "type" : it indicates the type of data retuned. In our case it's a"json"
      • "data" : in the post and put query it represents the data that should be imposed on the destination
    • The function(data) is defining what the code should do which the returned data if the request is successfull done
    • The function(error) is used if the HTTP query fails
