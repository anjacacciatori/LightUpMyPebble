(static) on()

This function defines what each selected item of the menu. This function is composed by a switch with 14 cases.

    • Case 0 and case 6 allows the user to select respectivly the light or the group that they want to modify
    • Case 1 and case 7 enables the user to swith respectivly the light or the lights of a group on or off
    • Case 2 and case 8 allows the user to change respectivly the light or group colors
    • Case 3 and case 9 allows the user to change respectivly the light or group saturation
    • Case 4 and case 10 allows the user to change respectivly the light or group brightness
    • Case 5 and case 11 allows the user to enable or disable the color loop
    • Case 12 allows to configure an authorized user
    • Case 13 searches the bridge IP address
